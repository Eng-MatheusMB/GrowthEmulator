⭐ Recursos

Por que GrowthEmulator?

Uma ferramenta completa para pesquisadores, engenheiros e estudantes de bioprocessos.

🔬

30+ Modelos Cinéticos

Exponenciais, logísticos, mecanísticos, sigmoides, enzimáticos e empíricos. De Malthus (1798) a Baranyi-Roberts (1994).

📈

Visualização Interativa

Gráficos Plotly com zoom, pan e hover. Curva ajustada em grade fina com banda de confiança ±1σ e análise de resíduos.

📁

Importação Inteligente

Suporte a .csv, .xlsx e .txt com detecção automática de delimitadores BR (;,) e EN (,,.). Entrada manual com Ctrl+V.

📐

Métricas Estatísticas

RMSE, R² Ajustado, AIC, BIC corrigido. Teste de Shapiro-Wilk / D'Agostino nos resíduos. Indicador de qualidade automático.

📄

Exportação Completa

Resultados em PDF (relatório de análise) e Excel (5 abas: Dados, Parâmetros, Métricas, Ajuste, Fases) prontos para publicação.

♿

Acessibilidade Total

Modo escuro/claro, alto contraste e paletas para daltonismo (deuteranopia, protanopia). Interface em PT, EN, ES e 中文.

🔬 Biblioteca

11 famílias de modelos

Cada família cobre uma classe de comportamento microbiano específico, com referências históricas e literatura validada.

🚀 Fluxo

Como funciona

Quatro passos simples do dado bruto ao resultado publicável.

1

📁 Carregar dados

Faça upload de .csv, .xlsx ou .txt, ou insira dados manualmente. Delimitadores BR/EN detectados automaticamente.

2

🔬 Escolher modelo

Explore os 11 boxes de modelos. Apenas os compatíveis com seus dados ficam ativos. Use o Guia de Seleção para ajuda.

3

⚙️ Ajustar & analisar

Configure tolerância, iterações e modo de restart. Execute o ajuste e visualize parâmetros, métricas, fases de crescimento.

4

📤 Exportar

Baixe PDF com relatório completo ou Excel com 5 abas de resultados. Pronto para periódicos e relatórios técnicos.

💻 Instalação local

Rodar localmente

Três comandos. Sem configuração adicional.

1 — Clonar o repositório

# via Git
git clone https://github.com/matheusmonteirobatista/growthemulator.git
cd growthemulator

2 — Instalar dependências

pip install -r requirements.txt

3 — Executar o simulador

streamlit run app.py

O simulador abrirá automaticamente em http://localhost:8501

📦 Requisitos mínimos

Python ≥ 3.9
Streamlit ≥ 1.32
Pandas ≥ 2.0
SciPy ≥ 1.11
Plotly ≥ 5.18
openpyxl + fpdf2
